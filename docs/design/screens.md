## コンポーネント一覧

| コンポーネント | 説明 | 使用画面 |
|--------------|------|---------|
| NavBar | ナビゲーションバー（ロール別メニュー表示） | 全画面 |
| ClockButton | 出勤/退勤ボタン（状態に応じて切替） | ダッシュボード |
| TimeRecordList | 打刻記録リスト | ダッシュボード, 勤怠一覧 |
| MonthSelector | 年月選択 | 勤怠一覧 |
| LeaveForm | 休暇申請フォーム | 休暇申請 |
| LeaveHistory | 休暇申請履歴テーブル | 休暇申請 |
| ApprovalCard | 承認/却下操作カード | 承認一覧 |
| StatusBadge | ステータスバッジ（承認済/申請中/却下） | 各一覧 |
| AttendanceTable | 勤怠テーブル（日別） | 勤怠一覧, 管理者勤怠 |
